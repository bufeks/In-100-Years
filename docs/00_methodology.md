# 方法論 — 未来から逆算して「いま安い必然」を見つける

このリポジトリの中心となる思考法。**100年後に必要とされるのに、いまは評価されていないもの**を体系的に探すための型を定義する。

---

## 0. 中心の問い

> **「自分は正しいと確信しているのに、ほとんどの人がまだ同意していない “未来の真実” は何か？」**

この問いに具体的に答えられたとき、そこに評価ギャップ（=安く仕込める余地）がある。賛同が多い未来はすでに価格に織り込まれている。狙うのは「正しいのに不人気」な一点。

---

## 1. 価値の式

```
期待リターン  ≈  不可避性  ×  評価ギャップ  ×  捕捉可能性
```

- **不可避性 (Inevitability)** — 100年スパンで、その需要・問題・変化はほぼ確実に来るか。物理・生物・人口・情報・人間性という「動かしにくい力」に根ざしているほど高い。
- **評価ギャップ (Mispricing)** — いま市場・社会がそれをどれだけ過小評価しているか。退屈・タブー・低ステータス・不可視・早すぎる、ほど大きい。
- **捕捉可能性 (Capturability)** — それが当たり前になったとき、自分が果実を握れる「所有」を築けるか。正しいだけでは1円も入らない。

3つの**積**であることが重要。どれかがゼロなら期待値はゼロに近い。

---

## 2. 候補を生む3つのレンズ

候補テーマは、次の3レンズを通して発生させる。3つすべてに引っかかるものが最良。

### レンズA：逆算（Backcasting）— 2125年の常識から逆向きに辿る

「100年後には当たり前」になっていそうな光景を、**具体的なワンシーンとして**描く。そこから現在へ橋を逆にたどり、**「それが成り立つために、誰かが先に作る／発見する／変える必要があるもの」**を抜き出す。

- 問い：「2125年、◯◯は当然になっている。では、その前提条件（ボトルネック）は何か？ それを最初に握るのは誰か？」
- 例：「2125年、人類のあらゆる知識は数千年劣化しない形で保存されている」→ 前提：超長期の物理的データ媒体 → ボトルネック：書き込み/読み出しの低コスト化 → 所有：その規格・素材・装置。

### レンズB：不可避な駆動力（Deep Drivers）— 100年でも折れない力を延ばす

10年の流行ではなく、**世紀をまたいでも方向が変わりにくい力**を起点にする。`02_domain-map.md` の各ドメインはこの駆動力に対応する。

- **物理** — 熱力学、エネルギー、材料、有限な元素（リン・希少金属）。
- **生物** — 老化、病、食、微生物、生殖。
- **人口** — 総数、出生率、高齢化、移動。
- **情報** — 計算、知識、信頼・真正性。
- **惑星の限界** — 気候、水、土壌、生態系。
- **人間性の定数** — 地位・承認、意味・孤独、死への恐れ、物語への欲求。

これらは「来るか来ないか」ではなく「いつ・どんな形で来るか」の問題。確実性が高いぶん、評価ギャップを探す価値がある。

### レンズC：評価ギャップ（Contrarian Gap）— いま見過ごされている場所を狙う

正しい未来でも、すでに人気なら安く仕込めない。次の「見過ごされやすさ」のサインを探す。

- **退屈／地味** — 肥料、配管、廃熱、保存。話題にならない＝安い。
- **タブー／低ステータス** — 死、介護、出生率、宗教、孤独。語りにくい＝参入者が少ない。
- **不可視** — 外部性・劣化・遅効性（土壌の枯れ、データの腐敗）。値段がついていない＝将来つく。
- **早すぎる記憶** — 過去に流行って一度死んだ領域（「あれは失敗した」と皆が思っている）。技術前提が変わって復活する。
- **辺境** — 主要メディア・主要市場の外（途上国の現場、ニッチな学会、規制の隙間）。

---

## 3. 時間アービトラージ（核心原理）

> **早すぎることは、間違っていることと区別がつかない。**

価値は「コンセンサスより前に確信を持つこと」に宿る。だからこのリポジトリの目的は、**世界が気づく前に確信を組み上げ、安いうちに所有を置く**こと。

- 「正しさ」より「タイミング × 確信 × 所有」。
- 早すぎて死なないために、レンズCで「いま小さく布石を打てるか（着手可能性）」も評価する。
- 確信は一度きりではなく、シグナルで更新し続ける（運用ループ）。

---

## 4. 「所有」をどこに置くか（捕捉の型）

正しい未来を当てても、所有の置き場所を間違えると儲からない。ゴールドラッシュの教訓：金を掘る人ではなく、つるはしを売る人が儲かった。置き場所の型：

| 型 | 説明 | 例 |
|----|------|----|
| 鉱山 | 希少な資源そのもの | 鉱区、土地、データ、権利 |
| つるはし | 全員が必要とする道具・部品 | 装置、素材、エネルギー、計算 |
| 関所 | 必ず通る場所・規格・標準 | プロトコル、認証、取引所 |
| 地主 | 価値が乗る土台・プラットフォーム | OS、ネットワーク、市場 |
| 知 | 真似されにくい知識・技術・人脈 | 特許、ノウハウ、専門家網 |

アイデアごとに「自分はどの型で握るか」を必ず1つ以上特定する。

---

## 5. 避けるべき失敗モード

- **確証バイアス** — 自分が好きな未来だけを「必然」と思い込む。→ 反証を必ず書く。
- **直線外挿** — S字カーブを直線で延ばす（早すぎ／遅すぎを生む）。
- **流行と必然の混同** — 10年のバズを100年の構造と取り違える。
- **捕捉不能の罠** — 正しいがオープンすぎて誰でも参入でき、利益が消える。
- **可視化済みの罠** — すでに皆が語っている＝織り込み済み。評価ギャップがない。
- **タイミング無視** — 100年後に必要でも、いま布石が一切打てないなら自分の人生では回収できない。

---

## 6. 運用ループ（再掲）

```
集める(signals) → つなぐ(trends/domains) → 採点(ideas + 評価表) → 布石(所有) → 反証で更新
```
