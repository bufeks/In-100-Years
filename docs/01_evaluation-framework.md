# 評価フレームワーク — アイデアの採点基準

`00_methodology.md` の価値の式（不可避性 × 評価ギャップ × 捕捉可能性）を、再現性のある採点に落とし込む。各アイデアはこの7軸で 1〜5 点をつけ、加重合計と確信ティアを出す。

---

## 採点の前に答える「殺しの質問」

採点する前に、次の4問に文章で答える。ここが甘いと点数は信用できない。

1. **なぜ、いまこれをやる人が少ないのか？**（評価ギャップの源泉はどこにあるのか）
2. **自分が間違っているとしたら、どこでか？**（最も強い反証は何か）
3. **これが当たり前になる前提条件（ボトルネック）は何で、誰がそれを握るのか？**
4. **自分はどの「所有の型」で果実を捕まえるのか？**（鉱山／つるはし／関所／地主／知）

---

## 7つの評価軸（各 1〜5 点）

| # | 軸 | 問い | 1点（低） | 5点（高） |
|---|----|------|-----------|-----------|
| 1 | **不可避性** | 100年スパンでその需要は来るか | 来るか不明・思いつき | 物理/生物/人口など動かしにくい力に根ざし、ほぼ確実 |
| 2 | **時間整合性** | 一過性でなく構造的・長期的か | 数年の流行 | 数十年〜100年かけて拡大する構造変化 |
| 3 | **評価ギャップ** | いまどれだけ過小評価されているか | 皆が注目し高値 | 退屈/タブー/不可視/早すぎで誰も見ていない |
| 4 | **文明規模** | 成立時のインパクト/市場の大きさ | ニッチ | 文明全体・全人類規模 |
| 5 | **捕捉可能性** | 防御できる「所有」を築けるか | 誰でも参入でき利益消失 | 強い堀（規格/特許/資源/ネットワーク） |
| 6 | **着手可能性** | 今日から布石を打てるか | 自分には手が届かない | 資本/知識/人脈で今すぐ小さく始められる |
| 7 | **非対称性** | 外れた時の損 vs 当たった時の利 | 外すと大損、当たって小利 | 外しても損は限定、当たれば青天井 |

> 各軸の「2」「4」は隣接段階の中間として運用。迷ったら低めにつける（楽観補正を避ける）。

---

## 合成スコア

### 加重合計

各軸に推奨ウェイトを掛けて合計（満点100）。

| 軸 | ウェイト |
|----|---------|
| 1 不可避性 | ×4 |
| 2 時間整合性 | ×3 |
| 3 評価ギャップ | ×4 |
| 4 文明規模 | ×3 |
| 5 捕捉可能性 | ×4 |
| 6 着手可能性 | ×1 |
| 7 非対称性 | ×1 |

`合成スコア = Σ(各軸点 × ウェイト)`（最大 = (5×4)+(5×3)+(5×4)+(5×3)+(5×4)+(5×1)+(5×1) = **100**）

> 不可避性・評価ギャップ・捕捉可能性を厚く重みづけしている。これが価値の式の3因子そのものだから。着手可能性と非対称性は「実行できるか／死なないか」のフィルタとして軽め。

### 確信ティア

| ティア | 合成スコア | 意味 |
|--------|-----------|------|
| **S** | 85〜100 | 最優先。積極的に所有を置きに行く |
| **A** | 70〜84 | 有望。深掘り＋小さな布石 |
| **B** | 55〜69 | 監視。シグナルを追い、条件が揃えば昇格 |
| **C** | 〜54 | 保留。記録は残すが今は動かない |

## クイック三角測量（採点前の足切り）

詳細採点の前に、**不可避性 × 評価ギャップ**の2軸で四象限に置いてみる。狙うのは右上だけ。

```
評価ギャップ 高
        │  【見送り】          │  ★★★ 本丸 ★★★
        │  正しくないかも       │  正しいのに誰も見ていない
        │  だが安い            │  （ここを深掘り）
────────┼─────────────────────┼──────────────  不可避性 高
        │  【無関係】          │  【織り込み済み】
        │  来ないし安い         │  正しいが皆が知ってて高い
評価ギャップ 低
```

右上（不可避性・高 × 評価ギャップ・高）に来たものだけ、7軸の本採点に進める。
